# Fib

Был разработан сайт ресторана с доставкой еды Fib.

Сайт написан на React с использованием TypeScript. Для управления состоянием используется Redux Toolkit. Сборка проекта осуществляется с помощью Webpack.

## Структура сайта

Сайт представляет собой одностраничное приложение (SPA) со следующими секциями:

- **Шапка сайта** — логотип, информация о доставке, номер телефона, кнопка корзины и адаптивное бургер-меню для мобильных устройств.
- **Слайдер** — промо-баннеры с поддержкой клавиатурной навигации и кастомными кнопками листания.
- **Новинки** — блок с карточками новых позиций меню.
- **Меню** — каталог блюд по категориям (Пицца, Паста, Супы, Салаты, Напитки, Десерты, Бакалея) с фильтрацией и добавлением в корзину.
- **Акции** — блок с промо-изображениями.
- **Оплата и доставка** — информация об условиях доставки и интерактивная карта с меткой адреса ресторана на основе Яндекс Карт.
- **Подвал сайта** — логотип, контакты, адрес, социальные сети, платёжные системы.
- **Корзина** — выезжающая боковая панель с добавленными блюдами, управлением количеством и итоговой суммой.
